Copy both `sub-detective.sh` and `sub-detective-jfilter.py` to the same directory and run `sub-detective.sh`.
If the host is register to the public console, you'll get some info about who and when the system was registered.
If the host is registered to a satellite, you'll get info about the satellite server, the org and other basics.

This is primarily useful for admins that inherited a host and need simple info about how the host is configured for updates.
